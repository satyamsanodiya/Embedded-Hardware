## 🚀 Active Project: Centralized 14S Battery Management System (BMS)
* **Target Specifications:** 14-cell series (~51V nominal), 60A maximum continuous current.
* **Architecture:** Texas Instruments (TI) based Analog Front End (AFE) paired with a low-cost 32-bit microcontroller.
* **Design Challenges Tracked:**
    * Optimizing high-side switching control circuits.
    * Ensuring robust isolation for CAN-based communication interfaces.
    * Designing thermal dissipation layouts for 60A current loops.
