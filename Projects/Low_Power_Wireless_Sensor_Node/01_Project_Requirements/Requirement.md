# Project Requirements

## Design Goal

The objective of this project was to create a low-power wireless sensing platform capable of operating from a rechargeable battery while supporting multiple environmental sensors.

The system should transmit sensor information wirelessly and provide future expansion capability.

---

## Functional Requirements

### Power Requirements

- Operate from Li-Ion battery
- Battery capacity range:
  - 500mAh to 2500mAh
- USB Type-C charging support
- Operation from USB power even without battery

---

### Sensor Support

The system shall support:

- Temperature Sensor
- Humidity Sensor
- CO2 Sensor
- Air Flow Sensor
- PIR Motion Sensor

---

### Communication Requirements

- BLE Wireless Communication
- External I2C Expansion Port

---

### User Interface

- RGB Status Indicator
- SPI Based Color Display

---

## Design Constraints

- Compact PCB Size
- Low Power Consumption
- Rechargeable Battery Support
- Low Cost Implementation
- Expandable Architecture
